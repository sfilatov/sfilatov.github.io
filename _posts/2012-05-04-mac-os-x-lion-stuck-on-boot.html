Symptoms:

If on startup your Mac shows Apple logo plus progress wheel for a long time.

  • Boot in verbose mode (CMD+V on boot)
  • You should see BootCacheControl missing /var/db/BootCache.plist
  • Last message is 'DSMOS has arrived'

Solution:

  • Load in single mode (CMD+S on boot)
  • Load root console
  • Remount partion for writing: "mount -uw /"
  • Fix ACL problems: "chmod -R -N /Volumes"
  • Reboot.
