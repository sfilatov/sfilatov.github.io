Symptoms:

If on startup your Mac stucked on Apple and progress wheel. In verbose mode (cmd+v on boot)

  • you see BootCacheControl missing /var/db/BootCache.plist
  • last message is 'DSMOS has arrived'

Solution:

  • Load in single mode (cmd+s on boot)
  • Load in root console.
  • Remount partion for writing: "mount -uw /"
  • Fix ACL problems: "chmod -R -N /Volumes"
  • Reboot.
