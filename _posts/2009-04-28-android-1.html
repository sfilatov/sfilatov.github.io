Все достаточно просто. Вся разработка на платформе ведется пока только с помощью программирования на Java притом не на Java SE или Java ME. Android это особенный подвид Java платформы со своим особенным API.
Следует сразу отметить качественную и продуманную объектную архитектуру основных частей всех приложений.
Один из основных постулатов платформы Android гласит, что все приложения написанные вами равны приложениям написанным остальными поставщиками. То есть вы можете заменить стандартные приложения поставляемые с устройством.

1) Для начала разработки следует скачать Eclipse 3.4 Ganymede. Притом я советую ставить версию eclipse не содержащую лишних плагинов. страница для скачивания. Следует скачать Eclipse for Java Developers (85MB).
К вопросам почему eclipse: потому что плагин для разработки к нему поддерживает сам Google и он гораздо качественне плагинов под IDEA и Netbeans.

2) После скачивания eclipse распакуйте его в отдельную папку и всё. Так он инсталлируется. После чего следует его запустить.

3) После установки и запуска Eclipse на него можно установить плагин ADT (Android Development Tools) от Google. Инструкции по установке могут менятся и их можно найти на сайте developer.android.com. Приведу краткое описание:
В eclipse выбрать Help > Software Updates....
Выберете Available Software. Нажмите Add Site
В диалоге укажите репозиторий для плагина https://dl-ssl.google.com/android/eclipse/

После чего поставте галочку и нажмите Install.
Плагин скачается и установится после чего перезапустите Eclipse.

4) Eclipse установлен и настроен теперь можно скачать и само SDK. На данный момент уже доступно SDK версии 1.5 (Cupcake). В нем возможен выбор целевой версии платформы 1.1 или 1.5. Советую скачать 1.5 ибо все приложения в итоге будут портированы на неё.
Вот страница для скачивания здесь

5) После установки SDK в настройках Eclipse стоит указать место его инсталяции (Window -> Preferences). После чего можно создать пустой проект, выбрав его из шаблонов и запустить emulator.

Отмечу что вместе с sdk поставляется полная документация по платформе и куча примеров. На эмуляторе уже установлены все примеры и их можно запустить и посмотреть как это все работает.
