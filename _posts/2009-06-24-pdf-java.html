--- layout: post title: "Работа с PDF на Java" date: '2009-06-24T02:25:00.001-07:00' author: "Степан" tags: modified_time: '2009-06-24T02:25:57.841-07:00' blogger_id: tag:blogger.com,1999:blog-6563657311735677139.post-3851231207505328730 blogger_orig_url: http://sfilatov.blogspot.com/2009/06/pdf-java.html --- Среди библиотек для работы с PDF на Java наиболее доступными для использования являются библиотеки:
1. JPX
2. jPod
3. PDF Clown

В данный момент стабильная версия JPX доступна для скачивания и использования в Java SE приложениях.
jPod также доступен для скачивания с SourceForge.

PDF Clown с полными исходными кодами доступен, но только в версии 0.0.7 Alpha.

Мне больше понравился PDF Clown, так как в нем реализованы различные уровни абстракции для интеграции библиотеки в приложение, а также полностью открыт исходный код. Таким образом велика вероятность, что её можно портировать на Android. Есть реализация на C#, что делает возможным использование библиотеки на Windows Mobile.
