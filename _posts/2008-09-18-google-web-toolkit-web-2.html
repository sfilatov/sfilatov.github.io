GWT совсем новая технология, в основе которой лежит компиляция Java кода в javascript. Скомпилированный javascript одинаково хорошо работает на всех современных популярных браузерах.
GWT - это возможность быстро и качественно писать front-end браузерные приложения без знания всех особенностей javascript.
