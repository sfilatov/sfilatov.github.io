--- layout: post title: Light Weight UI Toolkit date: '2008-09-19T13:07:00.000-07:00' author: "Степан" tags: - Java - LWUIT - JME modified_time: '2008-09-25T21:42:36.434-07:00' blogger_id: tag:blogger.com,1999:blog-6563657311735677139.post-5908143964443333141 blogger_orig_url: http://sfilatov.blogspot.com/2008/09/light-weight-ui-toolkit.html --- Сегодня заглянул на сайт Sun и случайно увидел видео про LWUIT. Очень интересная штука, учитывая что стандартные средства для разработки UI для JME приложений достаточно скудны, и конечно уступают по красоте нативным интерфейсам. Появление LWUIT - равносильно появлению Swing в Java после AWT. Современные возможности Swing на java 1.6 очень обширны в плане красоты и анимации.
Установил демо приложение на свой sony ericsson - впечатляет. Сравнимо по красоте с тем и интерфейсами, которые "вручную" прорисовываются в мобильных играх и приложениях. Есть даже анимация(поворот куба, появление меню сверху, сбоку, пропадание, диалоги).
Главное в этом всем, что это работает на всех телефонах с MIDP.
